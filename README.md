
#Neo

Neo是一个基于JDBC开发的，采用Hikaricp作为连接池的Orm框架。开发源头，源于几个原因，一个是之前几个公司采用的都是各自单独开发的Orm框架，另外一个是mybatis确实感觉不是很好用，把很多简单的东西设计的很复杂，最后是由于自己有很多想法，比如sql的规范落入到框架中等等，在之前接触的一些Orm框架中都没有。因此就有想法设计一个符合自己想法的Orm框架。
